# Kalori Takip Uygulaması

C# console uygulaması olarak geliştirdiğim kalori takip ve egzersiz hesaplama projesi.

## Özellikler
- BMR (bazal metabolizma hızı) hesaplama
- Günlük kalori ihtiyacı hesaplama
- Hareket durumuna göre kalori hesaplama
- Kardiyo ve ağırlık egzersizleri
- Egzersize göre yakılan kalori hesabı
- Günlük hedef belirleme
- Toplam yakılan kaloriyi takip etme
- Kullanıcı bilgisi doğrulama işlemleri

## Kullanılan Teknolojiler
- C#
- .NET
- Console App
- OOP mantığı
- List kullanımı
